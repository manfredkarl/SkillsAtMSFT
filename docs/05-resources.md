# Resources & Skill Repos
{: .fs-8 }

Curated skill repositories to supercharge your Copilot CLI setup.
{: .fs-5 .fw-300 }

---

## Skill Repositories

### Anthropic Skills

A growing collection of high-quality foundational skills including `skill-creator`, `pptx`, and more. This is the go-to repo for getting started.

🔗 [github.com/anthropics/skills](https://github.com/anthropics/skills/tree/main)

---

### Vercel Skills (skills.sh)

The open agent skills marketplace. Browse, search, and install community-contributed skills for frontend design, deployment workflows, and more.

🔗 [skills.sh](https://skills.sh/)

---

### Microsoft: Awesome Copilot Skills

A curated list of skills specifically designed for GitHub Copilot - from code review to CI/CD debugging to Azure workflows.

🔗 [github.com/github/awesome-copilot/skills](https://github.com/github/awesome-copilot/tree/main/skills/)

---

### Microsoft Skills

Microsoft's official skills repository with enterprise-focused skills for Azure, GitHub, and Microsoft technologies.

🔗 [github.com/microsoft/skills](https://github.com/microsoft/skills)

---

### Microsoft: Copilot for Azure

A large collection of Azure-specific skills - infrastructure provisioning, resource management, cost optimization, and more. One of the most popular skill sets on skills.sh.

🔗 [github.com/microsoft/github-copilot-for-azure](https://github.com/microsoft/github-copilot-for-azure)

---

### Microsoft: Azure Skills

Deep Azure skills covering services like Azure Functions, Cosmos DB, App Service, AKS, and more. Complements the Copilot for Azure repo with service-specific expertise.

🔗 [github.com/microsoft/azure-skills](https://github.com/microsoft/azure-skills)

---

### Skills Hub

A community-curated catalog for discovering and browsing Copilot skills. Search by category, popularity, or use case.

🔗 [github.com/samueltauil/skills-hub](https://github.com/samueltauil/skills-hub)
